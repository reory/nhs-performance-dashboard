# 😁Contributing to NHS Performance Dashboard

Thanks for your interest in improving this clinical dashboard! To keep the code clean and the clinical logic safe, please follow these simple guidelines:

## 🌿 How to Contribute
1. **Fork the Repo**: Create your own copy to work on.
2. **Create a Branch**: Use a descriptive name (e.g., `fix-validation-bug` or `add-search-feature`).
3. **Keep Logic Safe**: If you change anything in `app/models.py` or `clinical_logic.py`, ensure the tests still pass.
4. **Run Tests**: Always run `pytest` before submitting a Pull Request.
5. **Submit a PR**: Describe your changes clearly and link any related issues.

## 📄Reporting Issues
If you find a bug (especially in the RTT calculation or NHS number validation), please open an Issue with:
* A description of the bug.
* Steps to reproduce it.
* The expected vs. actual result.

**Happy coding** 😁!
